<a name="how-do-i-associate-ajax-client-behavior-with-an-aspnet-server-control"></a>[Gewusst wie:] Ein Steuerelement von ASP.NET zuordnen AJAX-Clientverhalten?
====================
durch [Chris PEL-Spareinlagen](https://twitter.com/chrispels)

Informationen Sie zum Erstellen eines Clientverhaltens basierend auf der Microsoft AJAX, die ein ASP.NET-Serversteuerelement auf eine ASPX-Seite zugeordnet werden können. Das Clientverhalten erweitert die Funktionalität von der standardmäßigen ASP.NET-Steuerelements über Zuordnung anstatt durch ein neues benutzerdefiniertes Steuerelement erstellen. Aus diesem Grund kann das Verhalten in mehreren Instanzen oder Typen von ASP.NET-Steuerelementen abhängig von der Implementierung wiederverwendet werden.
