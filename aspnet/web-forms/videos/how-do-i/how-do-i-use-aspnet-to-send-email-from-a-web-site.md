<a name="how-do-i-use-aspnet-to-send-email-from-a-web-site"></a>[Gewusst wie:] Verwenden von ASP.NET zum Senden von e-Mail-Adresse von einer Website
====================
durch [Chris Pels](https://twitter.com/chrispels)

In diesem video Chris Pels zeigt, wie zum Erstellen und Senden einer e-Mail in einer ASP.NET-Website. Erfahren Sie zunächst die Kernklassen in dem System.NET.Mail-Namespace zum Senden von e-Mail-Nachrichten in einer ASP.NET-Website aus. Finden Sie unter Vorgehensweise: Erstellen Sie eine e-Mail-Nachricht mithilfe der MailMessage-Klasse, wie Sie die MailAddress-Klasse für die Absenderadresse und wie Sie eine umfasst oder mehr Adressen zu verwenden. Als Nächstes sehen wie e-Mail gesendet wird, mithilfe der SmtpClient-Klasse. Schließlich erfahren Sie, bis die zusätzlichen Optionen zum Konfigurieren der e-Mail-Nachricht wie CC, BCC, Priorität und Hinzufügen von Anhängen.
