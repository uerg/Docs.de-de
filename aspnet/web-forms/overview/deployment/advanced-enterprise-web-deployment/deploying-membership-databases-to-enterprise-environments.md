## <a name="what-are-the-issues-when-you-deploy-a-membership-database"></a><span data-ttu-id="0695b-111">Was sind die Probleme, wenn Sie eine Mitgliedschaftsdatenbank bereitstellen?</span><span class="sxs-lookup"><span data-stu-id="0695b-111">What Are the Issues When You Deploy a Membership Database?</span></span>

<span data-ttu-id="0695b-112">Wenn Sie eine Strategie für die Bereitstellung für eine Datenbank, entwickeln, ist zunächst, die Sie berücksichtigen müssen in den meisten Fällen welche Daten, die Sie bereitstellen möchten.</span><span class="sxs-lookup"><span data-stu-id="0695b-112">In most cases, when you devise a deployment strategy for a database, the first thing you need to consider is what data you want to deploy.</span></span> <span data-ttu-id="0695b-113">In einer Entwicklungs- oder Test-Umgebung empfiehlt es sich um Benutzerkontodaten zum vereinfachten testen schnell und einfach bereitzustellen.</span><span class="sxs-lookup"><span data-stu-id="0695b-113">In a development or test environment, you might want to deploy user account data to facilitate quick and easy testing.</span></span> <span data-ttu-id="0695b-114">In einer Umgebung Staging- oder produktionsumgebung ist es sehr unwahrscheinlich, dass Sie Benutzerkontodaten bereitstellen möchten.</span><span class="sxs-lookup"><span data-stu-id="0695b-114">In a staging or production environment, it's very unlikely that you'd want to deploy user account data.</span></span>

<span data-ttu-id="0695b-115">Leider führen Datenbankrollenmitgliedschaften ASP.NET einige spezielle Aufgaben, die diese Entscheidung zu treffen, die sehr viel komplizierter:</span><span class="sxs-lookup"><span data-stu-id="0695b-115">Unfortunately, ASP.NET membership databases introduce some specific challenges that make this decision a lot more complex:</span></span>

- <span data-ttu-id="0695b-119">Je nachdem, wie Ihre Mitgliedschaftsdatenbank konfiguriert ist kann der Mitgliedschaftsanbieter den Schlüssel des Computers zum Verschlüsseln von Kennwörtern, und speichern sie in der Datenbank verwenden.</span><span class="sxs-lookup"><span data-stu-id="0695b-119">Depending on how your membership database is configured, the membership provider may use the machine key to encrypt passwords and store them in the database.</span></span> <span data-ttu-id="0695b-120">In diesem Fall werden alle Benutzerdaten für das Konto, die Sie mit der Datenbank bereitstellen kann, die auf dem Zielserver nicht verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="0695b-120">In this case, any user account data you deploy with the database will become unusable on the destination server.</span></span> <span data-ttu-id="0695b-121">Aus diesem Grund ist das Bereitstellen von Benutzerkontodaten nicht unterstützt.</span><span class="sxs-lookup"><span data-stu-id="0695b-121">For this reason, deploying user account data is not a supported scenario.</span></span>
