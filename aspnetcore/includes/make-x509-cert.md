Unter Windows können selbstsignierte Zertifikate mit dem [PowerShell-Cmdlet „New-SelfSignedCertificate“](/powershell/module/pkiclient/new-selfsignedcertificate?view=win10-ps) erstellt werden. Ein nicht unterstütztes Beispiel finden Sie unter [UpdateIISExpressSSLForChrome.ps1](https://github.com/aspnet/Docs/tree/master/aspnetcore/includes/make-x509-cert/UpdateIISExpressSSLForChrome.ps1).

Unter macOS, Linux und Windows können Zertifikate mithilfe von [OpenSSL](https://www.openssl.org/) erstellt werden.
